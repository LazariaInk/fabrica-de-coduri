MySQL Command Line Client.

În timpul instalării serverului MySQL, se instalează și un client în linia de comandă pentru lucrul cu bazele de date. De exemplu, pe Windows, puteți găsi programul MySQL 8.0 Command Line Client în meniul Start. Acesta este clientul propriu-zis în linia de comandă.

În plus, clientul este instalat în două variante imediat - cu suport Unicode și fără.

Hai să pornim MySQL 8.0 Command Line Client - Unicode. Inițial, veți fi solicitat să introduceți parola:

Aici trebuie să introduceți parola stabilită la instalarea MySQL pentru utilizatorul root. După conectarea reușită, veți putea trimite comenzi către server folosind acest client în linia de comandă.

Pentru început, să vedem ce baze de date există pe serverul implicit. Pentru aceasta, introduceți comanda show databases;.

După execuția acestei comenzi, vom vedea că pe serverul implicit deja există o serie de baze de date care îndeplinesc funcții administrative.

Acum, să creăm o bază de date folosind următoarea comandă SQL:


create database test;

Pentru a crea o bază de date, se utilizează comanda create database, urmată de numele bazei de date. În acest caz, baza de date va fi denumită test.

Pentru a interacționa cu o anumită bază de date, trebuie să o setați ca bază de date curentă. Pentru aceasta, executați comanda use, urmată de numele bazei de date. De exemplu, pentru a te conecta la baza de date test creată anterior, introduceți următoarea comandă: 


use test;

Apoi vom crea în această bază de date o tabelă folosind comanda:


create table users (id integer auto_increment primary key, name varchar(30), age integer);

Această comandă creează o tabelă numită users, în care vor exista trei coloane - id, name și age. Coloana id va stoca un identificator numeric unic pentru utilizator, generat automat de către baza de date. Coloana name va stoca numele utilizatorului, iar coloana age va stoca vârsta acestuia.

După aceasta, putem adăuga și obține date din tabela pe care am creat-o. Începem prin a adăuga o înregistrare în tabel folosind următoarea comandă:

insert into users (name, age) values ('Tom', 34);

Și în final, vom obține datele adăugate:


select * from users;

Astfel, în linii generale, putem lucra cu clientul în linia de comandă MySQL, MySQL Command Line Client.
