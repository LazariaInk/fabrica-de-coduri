Formate JSON

Lucrul cu JSON

JSON (JavaScript Object Notation) este un format ușor pentru stocarea datelor. JSON descrie structura și organizarea datelor JavaScript. Simplitatea JSON a făcut ca în prezent să fie cel mai popular format de transfer al datelor în mediul web, detronând alt format cândva popular, xml.

Obiectele JSON sunt foarte asemănătoare cu obiectele JavaScript, mai ales că JSON este un subansamblu al JavaScript. În același timp, este important să le distingem: JavaScript este un limbaj de programare, în timp ce JSON este un format de date.

JSON suportă trei tipuri de date: valori primitive, obiecte și mase (array-uri). Valorile primitive reprezintă șiruri standard, numere, valoarea null, valori logice true și false.

Obiectele reprezintă un set de date cel mai simplu, alte obiecte și mase. De exemplu, un obiect JSON tipic:

{

   "name": "Tom",

   "married": true,

   "age": 30

}

În JavaScript, acestui obiect i-ar corespunde următorul:

const user = {

   name: "Tom",

   married: true,

   age: 30

}

Cu toate acestea, există și diferențe: în JSON, numele proprietăților sunt închise în ghilimele duble, ca șiruri obișnuite. În plus, obiectele JSON nu pot stoca funcții, variabile, ca obiectele javascript.

Obiectele pot fi complexe:

{

   "name": "Tom",

   "married": true,

   "age": 30,

   "company": {

       "name": "Microsoft",

       "address": "USA, Redmond"

   }

}

Masele (array-urile) în JSON sunt similare cu masele javascript și pot, de asemenea, să stocheze date simple sau obiecte:

["Tom", true, 30]

Masă de obiecte:

[{

   "name": "Tom",

   "married": true,

   "age": 30

},{

   "name": "Alice",

   "married": false,

   "age": 23

}]


Serializarea în JSON și deserializarea

Pentru lucrul cu formatul JSON în limbajul JavaScript este destinat obiectul JSON. Acesta permite să transforme un obiect JavaScript în format json și invers.

Pentru serializarea obiectului javascript în json se utilizează funcția JSON.stringify():

// obiect javascript

const user = {

   name: "Tom",

   married: false,

   age: 39

};

// obiect json

const serializedUser = JSON.stringify(user);

console.log(serializedUser); // {"name":"Tom","married":false,"age":39}

Pentru operația inversă - deserializarea sau parsarea obiectului json în javascript se utilizează metoda JSON.parse():

const user = {

   name: "Tom",

   married: false,

   age: 39

};

// serializare

const serializedUser = JSON.stringify(user);

// deserializare

const tomUser = JSON.parse(serializedUser);

console.log(tomUser.name); // Tom
