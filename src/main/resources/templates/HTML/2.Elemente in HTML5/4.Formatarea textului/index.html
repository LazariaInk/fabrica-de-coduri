Formatarea textului

Unele elemente HTML sunt destinate pentru formatarea conținutului text, de exemplu, pentru a evidenția textul cu bold sau italic etc. Să examinăm aceste elemente:

  • <b>: evidențiază textul cu bold
  • <del>: taie textul
  • <i>: evidențiază textul cu italic
  • <em>: evidențiază textul cu italic, oferind textului o semnificație de importanță
  • <s>: tăie textul
  • <small>: face textul puțin mai mic decât cel din jur
  • <strong>: evidențiază textul cu bold, având o funcție de evidențiere logică pentru a arăta importanța textului
  • <sub>: plasează textul sub linia de bază
  • <sup>: plasează textul deasupra liniei de bază
  • <u>: subliniază textul
  • <ins>: definește textul inserat sau adăugat
  • <mark>: evidențiază textul, conferindu-i o nuanță de importanță

Să aplicăm toate aceste elemente:

<!DOCTYPE html>
<html>
   <head>
       <meta charset="utf-8">
       <title>Formatarea textului în HTML5</title>
   </head>
   <body>
       <p>Formatarea în <mark>HTML5</mark></p>
       <p>Acesta este textul <b>evidențiat</b></p>
       <p>Acesta este textul <strong>important</strong></p>
       <p>Acesta este textul <del>tăiat</del></p>
       <p>Acesta este textul <s>invalid</s></p>
       <p>Acesta este textul <em>important</em></p>
       <p>Acesta este textul <i>cu italic</i></p>
       <p>Acesta este textul <ins>adăugat</ins></p>
       <p>Acesta este textul <u>subliniat</u></p>
       <p>X<sub>i</sub> = Y<sup><small>2</small></sup>Z<sup><small>2</small></sup></p>
   </body>
</html>
