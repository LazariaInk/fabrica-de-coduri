Elementul fieldset și legend

Pentru gruparea elementelor de formular este adesea utilizat elementul fieldset. Acesta creează o graniță în jurul elementelor încapsulate, formând practic un grup din acestea. Împreună cu acesta se folosește elementul legend, care stabilește un titlu pentru grupul de elemente:

<!DOCTYPE html>
<html>
   <head>
       <meta charset="utf-8">
       <title>Elementele formei în HTML5</title>
   </head>
   <body>
       <h2>Accesul pe site</h2>
       <form>
           <fieldset>
               <legend>Introduceți datele:</legend>
               <label for="login">Login:</label><br>
               <input type="text" name="login" id="login" /><br>
               <label for="password">Parolă:</label><br>
               <input type="password" name="password" id="password" /><br>
               <input type="submit" value="Autentificare">
           </fieldset>
       </form> 
   </body>
</html>

La nevoie, putem crea pe un singur formular mai multe grupuri folosind elemente fieldset.
