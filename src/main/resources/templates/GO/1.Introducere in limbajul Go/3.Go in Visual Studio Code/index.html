Go în Visual Studio Code

Visual Studio Code este un editor de text avansat, multiplatformă și ușor, dezvoltat de compania Microsoft, care oferă evidențiere a sintaxei, sugestii inteligente pentru diverse limbaje de programare și multe alte funcționalități. Să vedem cum putem folosi acest editor pentru dezvoltarea în limbajul Go.

În primul rând, vom instala acest editor de text. Instalatorul pentru sistemul de operare dorit (există suport pentru Windows, Mac OS și Linux) poate fi găsit la adresa: https://code.visualstudio.com/.

После установки Visual Studio Code по умолчанию не имеет никакой поддержки языка Go. Поэтому необходимо установить соответствующее расширение. Для этого перейдем в Visual Studio Code в меню View -> Extensions

După instalarea extensiei, vom stabili pe hard disk un folder pentru stocarea fișierelor proiectului și vom deschide acest folder în Visual Studio Code. Folderul poate fi deschis prin meniul File -> Open Folder. Apoi, vom crea în VS Code un fișier nou, pe care îl vom numi main.go.

Deschidem fișierul main.go și introducem în el următorul cod:

package main
 
import "fmt"
 
func main() {
     
    fmt.Println("Hello Go de la fabricadecoduri.com")
}

Salvăm codul introdus apăsând combinația de taste Ctrl+S.

La lucrul cu fișiere Go în VS Code pot apărea diverse mesaje referitoare la necesitatea instalării unor pluginuri suplimentare. De exemplu, în bara de stare din partea de jos a ferestrei poate apărea mesajul "Analysis Tools Missing". Putem face clic pe acest mesaj, iar în partea de sus a ferestrei VS Code va fi afișată o listă de pachete recomandate pentru instalare. Pentru a le instala, este suficient să apăsăm pe butonul Install.

Un avantaj al VS Code este faptul că acest editor are un terminal integrat. Deschidem terminalul din meniu prin View -> Integrated Terminal. După aceasta, în partea de jos a ferestrei VS Code se va deschide terminalul încorporat. În mod implicit, acesta va fi poziționat în folderul curent al proiectului.

Introducem în terminal următoarea comandă și apăsăm Enter:

go run main.go
